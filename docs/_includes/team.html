Pseudo x86 assembler

Class project

In my computer architecture class, I was tasked with creating a pseudo x86 ASM compiler over the course of the semester. Several assignments involved writing assembly code for this compiler as we added more and more functionality to it. Having to manage the stack manually so that I did not overwrite my code or overflow the stack really solidified for me how memory works. (No links or images exist of this project).
