Dynamic Paging Algorithm

Academic project

I wrote a dynamic memory paging algorithm for one of my final projects in my operating systems class. I created several interconnected algorithms designed to efficiently page programs (and parts of programs) in and out of memory while keeping the number of “misses” (page outs) to a minimum. I had to consider the different demands of each kind of possible program in mind and design around what would ensure the most efficient use of memory possible. This project pushed my critical thinking skills to the limit, and taught me a lot about how memory works behind the scenes, growing my Linux and C skills.

Pseudo x86 assembler

Academic project

In my computer architecture class, I was tasked with creating a pseudo x86 ASM compiler over the course of the semester. Having to manage the stack manually so that I did not overwrite my code or overflow the stack really solidified for me how memory works. The image shows a snippet of what the pseudo x86 looks like.
